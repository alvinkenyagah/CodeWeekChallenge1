# CodeWeekChallenge1

## Description

### Student Grade Generator
Prompts the User to enters result from 0-100 and they will get an output of the relevant grade in relation to the results enter.

### Speed Detector
User enters the value of the speed and if the speed is less than 70, it prints ‚ÄúOk‚Äù. Otherwise, for every 5 km/s above the speed limit (70), it give the driver one demerit point and print the total number of demerit points. If the driver gets more than 12 points, the output print: ‚ÄúLicense suspended‚Äù.

### Net Salary Calculator
User inputs their gross salary, the program calculates and outputs the user's Payee Tax, NHIF, NSSF Deductions and net salary.
